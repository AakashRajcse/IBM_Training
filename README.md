# IBM_Training

This repository contains hands-on **Data Analytics and Machine Learning projects** completed during **IBM Training**. The projects focus on analyzing real-world datasets, performing data cleaning, exploratory data analysis (EDA), visualization, and applying basic machine learning techniques to derive meaningful insights.

## 🔧 Tools & Technologies
- **Programming Language:** Python  
- **Data Analysis:** Pandas, NumPy  
- **Data Visualization:** Matplotlib.pyplot, Seaborn  
- **Databases:** SQL (MySQL)  
- **Statistics:** Descriptive Statistics, Probability  
- **Machine Learning:**  
  - Scikit-learn  
  - Linear Regression  
  - Logistic Regression  
  - Classification & Regression Models  
- **Development Tools:** Jupyter Notebook, Git, GitHub  

## 📂 Project Highlights
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Data visualization using **Matplotlib** and **Seaborn**  
- Statistical analysis for business insights  
- Implementation of basic machine learning models  

## 🎯 Objective
To build a strong foundation in **Data Analytics and Machine Learning** by working on practical projects and developing the ability to convert raw data into actionable insights for decision-making.
